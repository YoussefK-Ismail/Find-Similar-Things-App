# 🧒 Find Similar Things App for Kids

A fun and educational app that helps kids find similar words and objects using AI!

![App Demo](screenshots/Screenshot1.png)

## 🚀 Live Demo

👉 [Try the App](https://find-similar-things-app-3dnybyqbfsxmicbjetmsun.streamlit.app)

## 🎯 About The Project

This app uses word embeddings and similarity algorithms to help children
understand relationships between words and objects. Kids can type any word
and instantly find similar things!

## ✨ Features

- 🔍 Find similar words instantly
- 🎨 Clean and colorful kid-friendly interface
- ⚡ Fast results using Scikit-learn
- 📚 Educational tool for kids
- 🌈 Covers animals, fruits, vehicles, emotions and more

## 🛠️ Technologies Used

- **Scikit-learn** - TF-IDF Vectorizer & Cosine Similarity
- **Streamlit** - Web interface
- **Python 3.10+**

## 🚀 Installation

1. Clone the repository:
```bash
git clone https://github.com/YoussefK-Ismail/Find-Similar-Things-App.git
cd Find-Similar-Things-App
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
streamlit run app.py
```

## 💡 Usage

1. Type any word in the input box
2. Click **"Find Similar Things!"**
3. See the top 5 similar words with similarity percentage!

## 🎮 Example Words to Try

- 🐱 Animals: `cat`, `dog`, `lion`
- 🍎 Fruits: `apple`, `banana`, `mango`
- 🚗 Vehicles: `car`, `bus`, `train`
- 😊 Emotions: `happy`, `sad`, `excited`
- 🏃 Actions: `run`, `jump`, `swim`
